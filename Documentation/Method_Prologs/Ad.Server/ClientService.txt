Method Name ->
	GetHandler
Method Description ->
	return a new instance of our Entity Repository Handler, with the connection string gotten from the method GetConnectionString. The return type is a interface called IRepository, which tell users what methods are accessible.
Input ->
	None
Output ->
	None
Returntype ->
	IRepository
Exceptions ->
	None
Assumptions ->
	None
